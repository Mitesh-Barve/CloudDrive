# ☁️ CloudDrive – Secure File Storage System

## 🚀 Live Demo
🔗 https://myclouddrive.onrender.com

---

## 🚀 Overview
CloudDrive is a full-stack cloud storage application that allows users to securely upload, manage, and retrieve files from any device.

Built with a scalable architecture, it incorporates secure authentication, efficient file handling, and a seamless user experience through a modern, responsive interface.

The project demonstrates strong hands-on experience in full-stack development, including API design, state management, backend integration, and building production-ready web applications.

---

## ✨ Features
- 🔐 User Authentication (Login / Signup)
- 📂 Upload and manage files
- ⭐ Mark files as favorites
- 🕒 Recent files tracking
- 📥 Download files
- 🗑️ Delete files
- 📊 Storage usage tracking
- 🌙 Modern dark UI
